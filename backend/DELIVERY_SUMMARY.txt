API ENDPOINTS
================================================================================

Health Check
  GET /health
  Response: { status: "ok", timestamp: "2024-07-04T..." }

Start Execution
  POST /api/runs/:id/execute
  Auth: Bearer <JWT_TOKEN>
  Body: {
    command: "plan|build|review|judge|retry|full-cycle",
    provider: "claude|codex|gemini",
    roles: ["planner", "builder"],
    goal?: "Your goal here",
    context?: { ... }
  }
  Response: { runId, status, provider, message }

Cancel Execution
  POST /api/runs/:id/cancel
  Auth: Bearer <JWT_TOKEN>
  Response: { runId, status: "cancelled" }

List Providers
  GET /api/runs/providers
  Auth: Bearer <JWT_TOKEN>
  Response: { providers: ["claude", "codex"], count: 2 }

WebSocket
  ws://localhost:3001
  Message: { type: "execute"|"cancel", payload|runId }
  Receives: { type: "progress"|"complete"|"error", data }
